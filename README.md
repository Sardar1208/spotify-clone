# spotify-clone
This is the spotify clone made using react native and nativebase.

Steps to setup -
- Install dependencies
```bash
cd spotify-clone
yarn
```
- Run this using expo : 
```bash
yarn start
# Run directly on web
yarn web
# Run directly on Android
yarn android
# Run directly on iOS
yarn ios
```
